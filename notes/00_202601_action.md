nestjs总览、概述；
开始跑项目；


nvm use v21.1.0
将电脑的npmrc文件，改为公网未登录的配置；
脚手架安装、创建项目：
$ npm i -g @nestjs/cli
$ nest new demo-01

$ cd demo-01
$ npm run start
$ npm run start:dev // 监视文件中的更改

$ npm run start -- -b swc //? 使用 SWC 构建器,加快开发过程（构建速度快 20 倍）,文档：https://docs.nestjs.cn/recipes/swc

查看源码：
demo-01/
├── src/
│   ├── app.controller.ts      # 一个具有单个路由的基本控制器;处理HTTP请求
│   ├── app.controller.spec.ts # 控制器的单元测试。
│   ├── app.module.ts          # 应用程序的根模块。组织应用结构
│   ├── app.service.ts         # 一个具有单个方法的基本服务。核心业务逻辑
│   └── main.ts                # 应用程序的入口文件，它使用核心函数 NestFactory 来创建 Nest 应用程序实例。


添加swc配置：
npm i --save-dev @swc/cli @swc/core

nest-cli.json文件配置：
"compilerOptions": {
  "builder": "swc"
}
